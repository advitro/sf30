# Shift Grabber V9 — License Activation & First Use

## Activate Your License

### 1. Open the Extension
Click the **SG** icon in your Chrome toolbar.

### 2. Enter Your License Key
In the popup that opens:
- Find the **License Key** field
- Paste or type your license key exactly as provided (no extra spaces)

### 3. Turn It ON
- Flip the big **Shift Grabber** toggle to **ON**
- The extension will verify your key with the license server (takes 1–2 seconds)
- If valid, the status badge turns **green (LIVE)**

### 4. Done!
Open your AtoZ schedule page. The extension automatically detects it and starts polling for available shifts.

---

## What Happens Next

### Auto-Detection
Once enabled, Shift Grabber automatically:
- Detects when you open an AtoZ schedule tab
- Starts polling for open shifts in the background
- Claims shifts instantly when they appear
- Shows a HUD overlay with status and stats

### The HUD
A small status panel appears in the bottom-right corner of AtoZ pages:
- **LIVE** = actively polling and ready to claim
- **PAUSED** = temporarily stopped (press **P** to resume)
- **FAST** = override mode for burst claiming (press **Shift+O**)

### Keyboard Shortcuts (while on AtoZ)
| Key | Action |
|-----|--------|
| **P** | Pause / Resume polling |
| **Shift + O** | Toggle override (fast) mode |
| **Shift + H** | Show / Hide HUD |
| **R** | Reload all AtoZ tabs |
| **Shift + T** | Turbo mode (aggressive polling) |

## Advanced Settings
Click **Advanced settings →** in the popup to access:

### Date Tabs
Pre-open specific dates so the extension polls them automatically. Add dates and click **Open Now**.

### Blacklist Dates
Skip shifts on dates you don't want to work. Add blackout dates and click **Apply**.

### Subscription
- **Manage Billing** — update payment method or cancel
- **Upgrade to Pro** — unlock higher-tier features
