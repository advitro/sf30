# Shift Grabber V9 — Installation Guide

## What You Need
- Google Chrome (latest version recommended)
- Windows, Mac, or Linux computer
- Your license key (sent after purchase)

## Step-by-Step Installation

### 1. Unzip the Package
Download and extract the ZIP file to a folder on your computer. You should see:
```
shift-grabber-v9/
├── extension/      ← This is the extension folder
└── docs/           ← Help documents
```

**Important:** Keep the `extension/` folder somewhere safe. Do not delete it — Chrome loads the extension directly from this folder.

### 2. Open Chrome Extensions Page
- Open Google Chrome
- Type `chrome://extensions` in the address bar and press Enter
- Or click the menu (⋮) → More tools → Extensions

### 3. Enable Developer Mode
- Look for the **Developer mode** toggle in the top-right corner
- Turn it **ON** (it will turn blue)

### 4. Load the Extension
- Click the **Load unpacked** button that appears
- Navigate to and select the `extension/` folder from the unzipped package
- Click **Select Folder**

### 5. Pin the Extension
- The Shift Grabber icon (SG) should now appear in your Chrome toolbar
- If you don't see it, click the puzzle-piece icon (extensions menu) in the toolbar
- Find "Shift Grabber V9" and click the pin icon (📌) next to it

### 6. You're Installed!
Click the SG icon in your toolbar to open the control panel. Next step: [activate your license](README-Activate.md).

---

## Updating to a New Version
When a new version is released:
1. Download the new package and unzip it
2. Go to `chrome://extensions`
3. Find Shift Grabber V9 and click the **refresh** icon (↻)
4. Click **Load unpacked** again and select the new `extension/` folder
5. Your settings and license key will be preserved
